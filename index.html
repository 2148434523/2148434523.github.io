更新记录

电视家 1.5.7   点击下载

  • 新增 增加P2P功能，增加流畅度。
  • 新增 增加更多盒子的支持。
  • 新增 增加快速换台时显示节目单。
  • 新增 半点、整点报时。
  • 新增 设置中增加P2P功能开关。
  • 删除 因特殊原因删除了高清频道。

电视家 1.5.6

  • 优化 重大更新, 改善视频源流畅度。
  • 新增 使用server.txt自定义频道服务器。

电视家 安卓手机版 1.3.2a   点击下载

  • 新增 电视播放基本功能，进入后直接是全屏播放界面。
  • 新增 频道收藏功能。
  • 优化 在线更新频道。
  • 系统 需要Android 4.0或以上操作系统。

电视家 1.5.5h

  • 新增 增加视频源切换功能，并且可以记忆上次选中的视频源。
  • 新增 断线重连:电视家开机自动启动早于PPPOE拨号成功情况下，在拨号成功后，尝试重新播放视频。
  • 优化 适配乐视遥控器的频道上下按键, 修复乐视S40/S50 USB不能自定义的问题。

电视家 1.5.5

  • 新增 增加频道自定义功能。
  • 新增 增加在线更新频道功能。
  • 优化 优化播放流畅度，减少大小屏切换带来的爆音。
  • 新增 增加画面比例切换: 全屏拉伸/保持比例。
  • 新增 增加了一些高清频道。

电视家 1.5.4c 测试版

  • 新增 增加设置界面选择中英文语言。 @意哥
  • 新增 可在线更新切换频道图片（用户体验需改进）、动态获取频道图标，减小apk体积。
  • 优化 解决启动时节目单加载阻塞问题。
  • 优化 修复设置页进入时播放器后台继续播放。

电视家 1.5.4b 测试版

  • 优化 全志芯片解码器优化，减少视频切换时间，在我播盒子上测试有效。
  • 新增 在设置中增加"其他频道"开关，满足部分用户看台多的需求(此分类源不稳定)。
  • 修复 修复其他频道只能收藏一个的Bug。

电视家 1.5.4版

  • 优化 首页删除二维码,后续会放在设置页面中。-感谢: @boy5031
  • 优化 调整普通模式、关爱模式的字体大小。 -感谢: @狂飞 @马晓明
  • 新增 增加英文支持，当前为英文系统时，可以体现出来。 -感谢: @热心用户
  • 优化 退出焦点默认放在"取消"按钮上。 -感谢: @狂飞
  • 优化 删除"我的节目单"功能，将"我的最爱"修改为"我的收藏"。 -感谢: @boy5031

电视家 1.5.3版

  • 新增 增加"设置"功能，可以设置自动全屏开关，自启动开关。 -感谢: @yqiumi @小郑
  • 新增 增加关爱模式，加大字体大小，方便老年人使用电视家。可以在设置页中进行开关该模式。 - 感谢: @金＆安
  • 新增 增加开机自启动: 启动盒子后，自动启动电视家软件。 -感谢: @文子 @yqiumi
  • 优化 按键调整: 改用上下键换台 左右键调节音量。 - 感谢: @狂飞 @东海岸

电视家 1.5.1版

  • 新增 开启后自动全屏进入上一次退出前播放的频道。
  • 优化 删除"其他分类"，这个分类中的视频源主要来自于其他直播软件，清晰度、稳定性不够。

电视家 1.5版

  • 新增 增加对遥控器数字键的支持: 进入全屏后，可以通过遥控器数字键进行换台。 感谢: @狂飞
  • 新增 增加OK键唤出菜单: 全屏幕界面，按 菜单 或者 OK 键都可以唤出菜单。 感谢: @狂飞
  • 新增 增加高清频道: 主要增加乐视的几个高清自制频道，其他高清内容陆续添加中。 感谢: @狂飞 @让爱随风 @杀人卷了刃
  • 优化 增加了几个地方卫视频道。

电视家 1.4版

  • 优化 完善对Android Pad以及智能电视的支持。
  • 优化 完美适配乐视T1盒子以及摩托罗拉平板。
  • 新增 增加自动更新功能。

电视家 1.3测试版

  • 新增 增加对Web Socket/Web Service的支持，内部测试用。可以通过网页命令行进行换台、播放指定URL的功能。
